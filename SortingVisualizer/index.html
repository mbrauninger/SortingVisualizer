MIKE'S SORTING VISUALIZER
Welcome to Quicksort! Quicksort is an algorithm that works by taking an array, and assigning a “pivot” value to the first element in that array. There are also two values that the algorithm continually compares to the pivot, an “i” value, and a “j” value. The i value begins at the beginning of the array, and continues moving right until a value is found that is greater than the pivot value. The j value, on the other hand, begins at the last index of the array, and continues moving left until a value less than or equal to the value of the pivot is found. When the indices of the i value and j value have been found, the elements at those indices will switch places. This will continue to happen until values are found where the j index is either in the same place or farther left than the i index. When this is the case, the j element switches places with the pivot element, and the newly placed pivot element is in “sorted position,” meaning that every element to the left of it is lower in value, and every element to the right of it is greater in value. Then the array splits into two sub-arrays based on the remaining unsorted elements on each side of the sorted element. This process continues recursively, until the entire array is sorted! Cool, right? This visualizer will show you how that process works, and hopefully help you achieve a greater understanding of the algorithm. If you are really trying to understand how the algorithm works, set the speed to a slow setting. A key can be found under the blue sorting area. For a sort with more elements, expand the width of your browser and refresh the page. Do the opposite to get less elements. Enjoy!
